Bible Miner

How to use

This is a search engine using Information Retrieval techniques in order to search through the whole bible.

First select a Bible version, then create your search query using at least three terms i.e
"God created heaven earth". When you get results also you will be able to click on any specific verse and the new search query will be created using the clicked verse keywords.

Right now a more advanced version it's available at Bible Miner website. This includes biblical dictionaries search, automated summaries and context reading.
